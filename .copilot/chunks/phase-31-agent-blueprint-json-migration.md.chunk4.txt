### Response Format Migration

**Before** (XML + Markdown):

```xml
<thought>
Analysis and reasoning...
</thought>

<content>
## Analysis Report

### Section 1

- Item 1
- Item 2

```

**After** (XML + JSON):

```xml
<thought>
Analysis and reasoning...
</thought>

<content>
{
  "title": "Analysis Report",
  "description": "Structured analysis results",
  "steps": [...],
  "estimatedDuration": "1 hour",
  "risks": [...]
}
</content>
```
