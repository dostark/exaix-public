│                     KNOWLEDGE PHASE                                     │
│                                                                         │

│                     EXECUTION PHASE                                     │
│                                                                         │

│                                                                         │

│    → TASK_FULFILLMENT criterion: are all requirements addressed?        │
│    → REQUEST_UNDERSTANDING criterion: does output match intent?         │
│    → Generic criteria: code correctness, security, style (unchanged)    │
