## Problem

After deploying a new workspace and submitting a request to the `senior-coder` agent via `exactl`, the request processing fails with `plan.validation.failed`. This occurs despite the daemon starting successfully and the agent apparently completing its execution.

## Reproduction Steps

```bash
# 1. deploy workspace (simulated)
./scripts/deploy_workspace.sh ~/Exaix

# 2. Configure and start daemon
cp exa.config.sample.toml exa.config.toml
exactl daemon start

# 3. Create test portal
cat >> ~/Exaix/exa.config.toml << EOF
[[portals]]
alias = "TestApp"
target_path = "/tmp/test-portal"
EOF

mkdir -p /tmp/test-portal/src
cd /tmp/test-portal
git init
echo "# Test App" > README.md
echo "export const version = '1.0';" > src/index.ts
git add . && git commit -m "Initial commit"
